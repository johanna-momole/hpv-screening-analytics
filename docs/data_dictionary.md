# Data Dictionary — HPV Screening Analytics Platform

All data in this project is **synthetic**, generated by the Faker library and
`src/generate_data.py`. No records represent real patients, providers, or
healthcare systems.

---

## Tables

### `insurance_type`
| Column | Type | Description |
|--------|------|-------------|
| `insurance_plan_id` | INTEGER (PK) | Surrogate key for the insurance plan |
| `plan_name` | TEXT | Human-readable plan name |
| `coverage_type` | TEXT | Category: Private, Medicaid, Medicare, Self-Pay, ACA, CHIP |
| `description` | TEXT | Free-text description |

## Derived Columns in `build_master()`

The function `src/metrics.build_master()` creates a flat analytical table
with the following additional derived columns:

| Column | Derivation |
|--------|-----------|
| `age` | `(scheduled_date − date_of_birth).days / 365.25` |
| `age_group` | Binned: 21–29, 30–39, 40–49, 50–64, 65+ |
| `year_month` | `scheduled_date` formatted as `YYYY-MM` |
| `year` | Year extracted from `scheduled_date` |
| `no_show` | 1 if `status == 'No-Show'`, else 0 |
| `completed` | 1 if `status == 'Completed'`, else 0 |
| `abnormal` | 1 if `result` ∈ {Positive, Abnormal}, else 0 |
